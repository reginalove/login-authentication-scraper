# Login Authentication Scraper

This Python script logs into a website and extracts protected content.

## Features

- Uses requests.Session for authentication
- Sends POST login request
- Maintains login session
- Accesses protected page
- Extracts content using BeautifulSoup
- Includes retry logic

## Technologies Used

- Python
- requests
- BeautifulSoup

## Purpose

Built to learn authentication handling and session-based web scraping.
